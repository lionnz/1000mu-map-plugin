# 📖 1000MU 3D 周边地图插件 (Figma to Blender) 

**版本**：v0.38  **维护者**：1000MU团队内部工具   **适用环境**：Blender 4.2.0 及以上

## 一、 核心功能与人工边界

本插件旨在通过自动化管线解决街区沙盘制作中的重复劳动。商业地产导视系统 3D 资产生产管线，实现从 Figma 2D 矢量到 WebGL 轻量化 3D 资产的 95% 自动化交付。

*   **自动化范围：** SVG数据清洗、物理定标、语义建模、UV 映射、Draw Call 优化。
    
*   **人工边界：** 针对 SVG 中存在的“0 面积面”（共线顶点），若导入后挤出异常，请通过「有限融并 (Limited Dissolve)」修复拓扑。
    

### 二、 黑魔法：技术原理深度解析

为了保证 1000MU 团队的资产交付质量，我们构建了以下 12 项自动化防线：

1.  **Figma 字符解码引擎**
    
    *   **问题：** Figma 导出的 SVG 经常将中文层名转码（如 `_x5546__x573a_`），导致 Blender 导入后层名乱码。
        
    *   **处理：** 内置逆向解码器，自动识别并还原西欧字符误读及十六进制编码，确保 Blender 中的图层列表完全等同于 Figma 设计稿的原始业务命名。
        
2.  **XML 标签烙印技术**
    
    *   **问题：** SVG 的编组结构（`<g>` 标签）在 Blender 导入后会被丢弃，导致层级丢失。
        
    *   **处理：** 在导入前对 SVG 进行预解析，将父层级的中文名称强行“烙印”到所有子级形状标签（`<path>`）的 `id` 属性中，实现层级结构的“扁平化映射”。
        
3.  **物理尺度绝对映射**
    
    *   **问题：** 2D 像素 (px) 到 3D 物理空间 (m) 的换算精度极低，模型导入后极易出现尺度错乱。
        
    *   **处理：** 设定团队标准比例 `10px=1m`，并推导固定乘数 `3543.0` 进行全域缩放，确保沙盘在 3D 软件中具备严格的物理真实尺寸。
        
4.  **视口居中算法**
    
    *   **问题：** Figma 原点在左上角，导致模型导入后总会偏移至远方，增加对齐成本。
        
    *   **处理：** 导入前计算画布视口 `viewBox`，通过前置偏移运算，强制将所有几何中心平移至 Blender 全局坐标原点 `(0,0,0)`。
        
5.  **属性冻结技术 (Early Transform Apply)**
    
    *   **问题：** 未重置缩放比例的物体在执行“实体化”挤出时，会产生指数级的几何形变。
        
    *   **处理：** 在执行挤出逻辑前，强制执行 `Apply Scale`，确保物理计算环境缩放系数固定为 `1.0`。
        
6.  **智能层级聚合**
    
    *   **问题：** Figma 的复杂编组导致导入后出现大量冗余的重名网格（如 `主路.001`、`主路.002`）。
        
    *   **处理：** 启用正则表达式过滤器，自动剥离 Blender 的系统后缀，将分散网格归并回业务基准命名，实现列表视图的精简。
        
7.  **图层预设配置、16进制色值转32位浮点**
    
    *   **问题：** 预设图层配置中缺少默认值，图层颜色为svg的16进制、转到3d时出现gamma异常。
        
    *   **处理：** 在插件代码中预设不同图层的默认挤出高度（如 `主路`、`绿化`）。在插件代码中增加16位颜色转32位浮点，自动配置到图层预设中。
        
8.  **Z-Fighting 防闪烁微增量**
    
    *   **问题：** 多层纯平面重叠时，WebGL 引擎因计算精度限制，产生闪烁现象。
        
    *   **处理：** 挤出引擎自动检测 0 厚度层级，按图层索引分配 `0.001m` 的微小 Z 轴阶梯，从物理层面物理隔离重叠面。
        
9.  **法线 Z-Lock 修复**
    
    *   **问题：** 不规则路径导致的法线反向，会让建筑向下长入地底。
        
    *   **处理：** 引入 `BMesh` 底层接口，在内存中遍历多边形法线，锁定朝向为天空（+Z），确保建筑生长方向绝对正确。
        
10.  **懒加载防假死引擎**
    
    *   **问题：** 强制下载拼音库会阻塞主线程，导致软件假死。
        
    *   **处理：** 将依赖下载剥离至后台异步，若未配置环境则自动启用“字符剔除降级模式”，确保操作全流程不卡顿。
        
11.  **显存级 UV 坍缩矩阵**
    
    *   **问题：** 传统 Python 循环修改顶点坐标，面对大型街区会触发 CPU 瓶颈。
        
    *   **处理：** 调用 C++ 级的 `foreach_set` 内存写入接口，直接操作内存缓冲区，实现 UV 数据的批量写入，效率提升百倍。
        
12.  **viewBox 自适应捕获**
    
    *   **问题：** SVG 头部定义若为百分比（`width="100%"`），直接读取会导致类型报错。
        
    *   **处理：** 建立异常捕获机制，读取失败时自动下穿解析 `viewBox` 参数，抓取真实视口尺寸进行兜底换算。
        
13.  **双轨制图集分流引擎**
    
    *   **问题：** 全局混合模式导致 WebGL 深度排序冲突，出现“远方物体覆盖前方”的现象。
        
    *   **处理：** 采用唯一色卡贴图，生成两套独立渲染材质（Opaque 轨道处理实体、Transparent 轨道处理透明水系），实现 Draw Call 最小化与物理遮挡绝对正确。
        

## 三、 标准工作流与浮窗操作指引

在 Blender 侧边栏 `1000Map` 面板中，核心功能已内置详尽的浮窗注释说明：

### 1. 📦 数据导入 (插件标签页 1)

*   **\[导入并重构空间\]：** 自动执行物理单位换算与绝对居中，智能嗅探 viewBox 尺寸，静默修复 Figma 中文乱码，并一键转换为网格平面。
    
### 2. 🏗️ 预设挤出 (插件标签页 2)

*   **\[随机高度\] 开关：** 开启后可设定区间，引擎将为同名图层体块随机分配高度，构建错落城市天际线。
    
*   **\[一键挤出\]：** 强制纠正法线朝上并按高度挤出，自动为纯平面附加防闪烁(Z-fighting)微缩阶梯，并赋予基础材质。
    
### 3. 📤 导出交付 (插件标签页 3)

*   **\[生成色块贴图\]：** 【双轨制引擎】提取色彩与 Alpha 通道生成全局图集。极速坍缩 UV 并自动分流实体与透明材质，将 Draw Call 降至极致。
    
*   **\[导出 GLB\]：** 一键导出符合 WebGL 标准的 .glb 模型。
    
*   **\[拼音防呆\] 选项：** 勾选后，导出时静默将中文图层名转换为拼音，防止 WebGL 引擎路径解析报错。
    

## 四、 设计团队制图规范

1.  **关键词命名：** 确保图层包含业务词（如：`绿化`、`主路`、`浅色box`），引擎将自动匹配高度规则。
    
2.  **透明度设置：** 水面、玻璃等组件，直接设置 Figma `Fill Opacity`，插件自动捕捉并生成 `Blend` 透明材质。
    
3.  **几何拓扑：** 建议在 Figma 内部合并形状 (`Union`)，确保矢量闭合无孤立点。

## 五、 安装指南

1.  打开 Blender（推荐使用 **Blender 4.2 或更高版本**）。
    
2.  点击顶部菜单栏 **\[编辑 (Edit)\] -> \[偏好设置 (Preferences)\]**。
    
3.  在弹出的窗口左侧点击 **\[插件 (Add-ons)\]**。
    
4.  点击右上角的 **\[从磁盘安装 (Install from Disk)\]**。
    
5.  在文档最下方下载 `1000MU_Map_Plugin_038.py` 文件并点击确认。
    
6.  blender**自动勾选开启**插件，在偏好设置左下角点击**保存用户设置**、或者开启**自动保存设置**
    
7.  回到 3D 主视图，按下键盘 `**N**` **键**，点击右侧全新的专属侧边栏标签 **\[1000Map\]**，即可开启高能工作流。
