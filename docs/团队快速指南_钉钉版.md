# 1000MU 3D 周边地图插件 · 团队快速指南

> 适用版本：v0.3.14 | 适用环境：Blender 4.2+

---

## 插件能做什么

把 Figma 导出的 SVG 矢量地图，一键变成 Blender 里的 3D 沙盘模型，并导出为 Web 端可用的 GLB 文件。

**核心流程**：`SVG 导入 → 配置参数 → 一键挤出 → 材质优化 → GLB 导出`

> v0.3.14 内置了拼音引擎，开箱即用，不需要联网装任何东西。

---

## 安装与升级

### 第一次安装

1. 打开 Blender，**编辑 → 偏好设置 → 插件**
2. 点击右上角 **从磁盘安装**
3. 选择 `1000MU_Map_Plugin_v0314.zip`，安装后勾选启用
4. 回到 3D 视图，按 `N` 键打开侧边栏，点击 **`1000Map`** 标签即可看到面板

### 升级（已安装旧版本）

1. 插件管理中先**禁用并卸载**旧版
2. 按「第一次安装」步骤装新版
3. 两个版本不能同时启用

## 操作流程（三标签页）

插件面板有 **导入 / 配置 / 导出** 三个标签页，按顺序操作即可。

### 标签页 1 · 导入

| 操作 | 说明 |
|---|---|
| 选择 SVG 文件 | Figma 导出的 `.svg`，插件会自动解码中文层名 |
| 设置比例 | 默认 `10px = 1m`（团队标准），可调整 |
| 曲线精度 | 默认 4，圆弧多的图可调到 6-8 |
| 点击「导入并重构空间」 | 自动完成：解码 → 换算尺寸 → 居中 → 提取颜色 |

导入完成后图层高度会自动从偏好设置预设表匹配。

### 标签页 2 · 配置

导入后会自动生成图层列表，每行可调整：

| 参数 | 说明 |
|---|---|
| 眼睛图标 | 关掉则不参与挤出 |
| 图层名 | 从 SVG 自动识别，只读 |
| 高度（m） | 按关键词自动匹配预设值，可手动改 |
| 随机高度 | 点击骰子图标开启，设置区间后逐个随机错落 |
| 颜色 | 从 SVG 提取，可手动覆盖 |

**配置好参数后，点击底部「一键挤出」**，插件会自动：

- 翻转朝下的法线使其朝上
- 按设定的高度挤出立体模型
- 焊接顶点 + 三角化（优化模型拓扑）
- 生成 Principled BSDF 材质
- 清理无用的孤立数据

### 标签页 3 · 导出

#### 步骤 A：生成色块贴图

点击 **「生成色块贴图」**，插件会：

- 把所有物体的材质颜色打包成一张 `3DMap_ColorAtlas` 贴图
- 自动识别透明材质，分流为不透明/透明两套图集材质
- 单物体多材质球也能正确处理，无需手动分离
- 合并重复材质槽，精简 GLB 导出列表

#### 步骤 B：导出 GLB

| 设置 | 说明 |
|---|---|
| 路径 | 默认 `~/Downloads` |
| 文件名 | 自动补 `.glb` 后缀 |
| 仅导出可见 | 跳过隐藏的物体 |
| 应用修改器 | 导出时烘焙修改器效果 |
| 拼音防呆 | 中文名自动转拼音，防前端报错（导出后会自动还原原名） |

点击 **「导出 GLB」** 完成。

## 自定义高度预设

**编辑 → 偏好设置 → 插件 → 展开「1000MU 3D周边地图插件」**：

- 表格中直接编辑「关键词」和「高度(m)」
- 点 `[+]` 添加新规则，`[-]` 删除当前行
- 修改自动保存，所有 `.blend` 文件共享
- 点「恢复内置默认值」可还原出厂设置

**匹配规则**：图层名「包含」关键词即命中。如 `主路-辅道` 命中 `主路` → 自动设为 20m。

内置默认值：

| 关键词 | 高度 | 关键词 | 高度 |
|---|---|---|---|
| 绿化 | 10m | 深色box | 30m |
| 主路 | 20m | 文字 | 0.5m |
| 支路 | 15m | 商场边框 | 60m |
| 中色box | 40m | 商场 | 80m |
| 浅色box | 50m | 水 | 5m |

## 注意事项

1. **操作顺序**：必须按 **导入 → 挤出 → 色块贴图 → 导出** 顺序执行，不能跳过
2. **拼音防呆**：v0.3.14 内置拼音引擎，开箱即用，无需额外安装
3. **多材质物体**：一个网格包含多个材质球无需手动分离，插件会逐面匹配色块并写入 UV
4. **颜色去重**：SVG 中同名图层出现不同颜色时，只保留首次出现的颜色
5. **拓扑修复**：如果挤出后模型异常（0 面积面/共线顶点），在 Blender 中使用「有限融并 (Limited Dissolve)」修复
6. **拼音防呆机制**：导出时临时改名 → 导出成功 → 自动还原原名，不污染 `.blend` 源文件
7. **法线方向**：插件会自动强制法线朝上（+Z），并且只会处理地图物体，不误伤场景其他资产

## 常见问题

**Q：导入后图层名是乱码？**
A：Figma 编码导致，已尽力还原。可在 Figma 侧用英文层名避免。

**Q：色块贴图提示"未找到材质"？**
A：请先完成"一键挤出"再执行色块贴图，按标签页顺序操作。

**Q：GLB 导出体积过大？**
A：降低曲线精度（标签页 1）、关闭不必要的图层（标签页 2 眼睛图标）、清理场景中无关网格。

**Q：场景里其他模型会被插件动到吗？**
A：不会。插件只会处理通过图层列表识别的地图物体。
