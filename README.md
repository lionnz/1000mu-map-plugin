# 1000MU 3D 周边地图插件 (Figma to Blender)

> 商业地产导视系统 3D 资产生产管线插件，将 Figma 2D 矢量地图一键转换为 WebGL 轻量化 3D 资产。

## 简介

本插件是 1000MU 设计团队内部工具，用于将 Figma 导出的 SVG 矢量地图自动转换为 Blender 3D 沙盘模型，并导出为符合 WebGL 标准的 GLB 文件。覆盖从 SVG 导入、物理尺度换算、语义化挤出、双轨制材质图集、到 GLB 导出的完整管线。

## 核心特性

- **SVG 智能导入**：自动解码 Figma 中文层名，物理尺度映射，全局居中
- **语义化挤出**：按图层关键词匹配默认高度，支持随机高度错落
- **双轨制图集引擎**：按材质聚合色彩，自动分流 Opaque/Transparent 材质
- **单物体多材质球支持**：按面写入 UV，无需分离物体，零顶点增量
- **冗余槽自动合并**：打包后无重复材质槽，GLB 材质列表精简
- **拼音防呆导出**：中文物体/材质名自动转拼音，防止前端报错

## 安装

1. 下载 [最新 Release](releases/) 中的 `1000MU_Map_Plugin_vXXX.py`
2. Blender → 编辑 → 偏好设置 → 插件 → 从磁盘安装
3. 选择下载的 `.py` 文件并启用
4. 按 `N` 键打开侧边栏，点击 `1000Map` 标签

> 详细安装与使用说明见 [技术说明书](docs/技术说明书_v0.3.11.md)

## 工作流

```
Figma 导出 SVG
       ↓
[导入] SVG 智能解码 + 物理换算 + 居中
       ↓
[预设] 图层高度配置 + 颜色校正
       ↓
[挤出] 法线修复 + Solidify + Weld + Triangulate + 材质生成
       ↓
[优化] 双轨制色块图集 + 按面 UV + 冗余槽合并
       ↓
[导出] GLB + 拼音防呆
```

## 团队规范

- 图层命名包含业务关键词（如 `绿化`、`主路`、`商场`）可自动匹配默认高度
- 透明物体（水面、玻璃）在 Figma 中设置 Fill Opacity 即可自动识别
- 业务命名放在 `<g>` 父级，子级形状会继承父级名
- 单个网格物体可包含多个材质球，无需手动分离
