# 变更日志

本文件记录 1000MU 3D 周边地图插件的所有版本变更。

---

## [v0.3.14] - 2026-07-08

### 重大变更
- **内置拼音引擎**：将 pypinyin 打包进插件（vendor），拼音防呆功能开箱即用，不再需要联网安装
- **移除 pip 安装机制**：删除 `MAP_OT_install_env` 操作符及导出页的拼音安装警告 UI
- **安装方式变更**：从单 .py 文件改为 .zip 包（因需内置 pypinyin 数据文件）

### 融合优化（v0.3.9 – v0.3.13）
- 兼容 Blender 4.5：`material.shadow_method` 改为 `hasattr` 兼容检测
- 偏好设置可配置高度预设表
- 色块贴图：支持单物体多材质球 + 冗余槽自动合并 + 回归修复
- 法线统一隔离、非递归孤立清理、epsilon 防抖、md5 防重名、tempfile 隔离
- 随机高度开启时自动填充区间、UI 布局优化、description 悬浮提示
- 新增 `_on_use_rand_height` 回调

---

## [v0.3.8] - 2026-07-07

### 初始版本
- 用户使用 AI coding 完成的初版插件。
- 核心功能：SVG 导入、物理换算、图层预设、挤出、基础材质生成、双轨制图集、GLB 导出。

### 已知问题（均已在 v0.3.14 中修复）
- 单物体多材质球时只贴首个材质
- 挤出末尾法线统一会误伤场景其他物体
- 递归孤立数据清理会误删用户保留资产
- 多 Blender 实例同时导入 SVG 会冲突
- 拼音功能需联网安装 pypinyin
